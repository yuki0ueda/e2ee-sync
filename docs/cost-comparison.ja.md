## なぜ e2ee-sync？

| | e2ee-sync | Dropbox / Filen / pCloud |
|---|----------|------------------------|
| **暗号鍵** | ユーザーが生成・保持。サービスに渡さない | サービスが管理（ゼロ知識でも信頼が必要） |
| **ストレージ** | R2, S3, B2, IDrive e2, Internxt 等から選択 | そのサービス固定 |
| **データの場所** | リージョン・プロバイダーを自分で選べる | サービスが決定 |
| **ベンダーロックイン** | なし。いつでも切替。データは標準S3オブジェクト | 完全ロックイン |
| **サービス終了リスク** | S3バケットは自分のもの。rcloneで直接アクセス可能 | サービス終了=データ移行必須 |
| **コスト（10GB）** | **無料**（R2/B2/IDrive e2 無料枠） | 無料枠小さい（Dropbox 2GB） |
| **コスト（100GB）** | **$0.40/月**（IDrive e2） | $1.67/月〜（Filen年払い） |
| **オープンソース** | MIT ライセンス。コード監査可能 | ほぼクローズドソース |
| **モバイル** | ❌ なし | ✅ iOS/Android |
| **Web UI** | ❌ なし | ✅ ブラウザアクセス |
| **セットアップ** | CLI（rclone + Tailscale 必要） | インストールのみ |

**e2ee-sync は「最も手軽」でも「すべてのケースで最安」でもない。「最も透明で自由な」E2EE 同期ツール。** 暗号鍵を自分で管理し、ストレージを自由に選べ、いつでもデータへのアクセスを失わない。

## 選択ガイド

```
10GB以下？
  → e2ee-sync + R2 or B2 or IDrive e2（無料）

10〜100GB？
  → e2ee-sync + IDrive e2（$0.40/月）or B2（$0.60/月）

100GB〜1TB、3年以上使う？
  → Internxt 買い切り（€195〜345）or Koofr（$130）
  → それ以外: e2ee-sync + IDrive e2（1TB $4/月）

1TB超、3年以上使う？
  → Internxt 5TB 買い切り（€495）→ 2年で元取れる
  → それ以外: e2ee-sync + IDrive e2（$4/TB/月）

モバイル・Web アクセスが必要？
  → Filen or Dropbox（e2ee-sync はデスクトップのみ）

暗号化を完全に自分で管理したい？
  → e2ee-sync（鍵をユーザーが保持する唯一の選択肢）
```
