# コスト比較

[English](cost-comparison.md) | [READMEに戻る](../README.ja.md)

e2ee-sync のクラウドストレージコストを、S3互換プロバイダー・サブスクサービス・買い切りプランと比較します。

> 2026年4月時点の価格。最新情報は各プロバイダーのサイトで確認してください。

## S3互換ストレージ（従量制）

e2ee-sync と S3 API で直接連携するプロバイダー。

| プロバイダー | ストレージ/GB | Egress | API | 最低利用 | 無料枠 |
|------------|-------------|--------|-----|---------|--------|
| **IDrive e2** | **$0.005** | 3x無料※1 | 無料 | **1TB** | 10GB |
| **Backblaze B2** | $0.006※3 | 3x無料※1 | 無料※4 | なし | 10GB |
| **Wasabi** | $0.007 | 1:1無料※2 | 無料 | **1TB** | なし |
| **Internxt S3** | €0.007 | 無料 | 無料 | **1TB** | なし |
| **Cloudflare R2** | $0.015 | **無料（無制限）** | $4.50/100万 | なし | 10GB |
| AWS S3 Standard | $0.023 | $0.09/GB | $5.00/100万 | なし | 5GB（12ヶ月） |

※1 保存量の3倍まで無料、超過分 $0.01/GB ※2 Wasabi: 保存量と同量まで、90日最低保持期間あり
※3 Backblaze B2: 2026年5月1日より $6.95/TB（$0.00695/GB）に改定予定
※4 Backblaze B2: 2026年5月1日より全顧客で標準APIコールが無料化（従来は無料枠超過で課金）
※ Internxt S3 は30日の最低保持期間あり

## なぜ e2ee-sync？

| | e2ee-sync | Dropbox / Filen / pCloud |
|---|----------|------------------------|
| **暗号鍵** | ユーザーが生成・保持。サービスに渡さない | サービスが管理（ゼロ知識でも信頼が必要） |
| **ストレージ** | R2, S3, B2, IDrive e2, Internxt 等から選択 | そのサービス固定 |
| **データの場所** | リージョン・プロバイダーを自分で選べる | サービスが決定 |
| **ベンダーロックイン** | なし。いつでも切替。データは標準S3オブジェクト | 完全ロックイン |
| **サービス終了リスク** | S3バケットは自分のもの。rcloneで直接アクセス可能 | サービス終了=データ移行必須 |
| **コスト（10GB）** | **無料**（R2/B2/IDrive e2 無料枠） | 無料枠小さい（Dropbox 2GB） |
| **コスト（100GB）** | **$0.60/月**（Backblaze B2） | €1.99/月〜（Filen Pro I, 200GiB） |
| **オープンソース** | MIT ライセンス。コード監査可能 | ほぼクローズドソース |
| **モバイル** | ❌ なし | ✅ iOS/Android |
| **Web UI** | ❌ なし | ✅ ブラウザアクセス |
| **セットアップ** | CLI（rclone + Tailscale 必要） | インストールのみ |

**e2ee-sync は「最も手軽」でも「すべてのケースで最安」でもない。「最も透明で自由な」E2EE 同期ツール。** 暗号鍵を自分で管理し、ストレージを自由に選べ、いつでもデータへのアクセスを失わない。

## 選択ガイド

```
10GB以下？
  → e2ee-sync + R2 or B2 or IDrive e2（無料）

10GB〜1TB？
  → e2ee-sync + Backblaze B2（最低課金なし、$0.006/GB）
  → IDrive e2 と Internxt S3 は1TB最低課金のため割高

1TB前後、2年以上使う？
  → Koofr 買い切り（$130, E2EE非対応）or Internxt 買い切り（セール頻繁）
  → それ以外: e2ee-sync + IDrive e2（1TB $5/月）

1TB超、2年以上使う？
  → Internxt 5TB 買い切り（定価€495、実勢はさらに安い）→ 2年未満で元取れる
  → それ以外: e2ee-sync + IDrive e2（$5/TB/月）

モバイル・Web アクセスが必要？
  → Dropbox, Filen, pCloud 等（e2ee-sync はデスクトップのみ）

暗号化を完全に自分で管理したい？
  → e2ee-sync（鍵をユーザーが保持する唯一の選択肢）
```
